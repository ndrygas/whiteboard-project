Whiteboard - Hackbright Capstone Project
by Nicolas Drygas

Project Description:

This whiteboard application aims to create a clean and efficient interface for
users to create and edit notes. These notes can also be shared between users 
as needed to provide a platform to collaborate on projects.

Tech Stack:

Languages: Python, Javascript, HTML, SQL
Frameworks and Libraries: Flask, Jinja, SQLAlchemy, Bootstrap
Database & Industry Tools: Command Line & PostgreSQL

Features:

-Users can create and log in to a personal account
-Each user can create, edit, and delete notes.
-Notes can also be shared between users and edited from all parties included 
in the share
